# Next.js + WooCommerce (Headless) Boilerplate (TypeScript)

This archive contains a minimal starter template (file structure + stub files) for a Next.js frontend connected to a headless WordPress/WooCommerce backend.

Features:
- Next.js (app router)
- TypeScript
- React Query
- Zustand (cart store)
- Axios clients for WooCommerce and WP
- API proxy routes examples
- Minimal pages and hooks (no CSS/visuals) — use your Figma design

Instructions:
1. Unzip and `cd` into the project folder.
2. `npm install` (or `yarn`)
3. Fill `.env` from `.env.example`
4. `npm run dev`

Files included: app/, lib/, hooks/, store/, components/, app/api/ routes.
